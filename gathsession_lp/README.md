### 📂 How to Run
1. _Clone the repository_:
```bash
git clone https://github.com/Lucaas9113/header-projects-non-responsive
```
2. _Navigate to the gathsession_lp folder:_
```bash
cd header-projects-non-responsive/gathsession_lp
```
 3. If using Vite, install dependencies and run:
 ```bash
npm install
npm run dev
```
